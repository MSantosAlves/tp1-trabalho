Detailed Description

Classe cuja função é armazenar uma string responsável por representar a data da Sessao.

Formato: DD/MM/AAAA Em DD tem-se indicação do dia: 01 a 31. Em MM tem-se indicação do mês: 01 a 12. Em AAAA tem-se indicação do ano: 2000 a 9999.
